01 — Overview

프로젝트 개요

SubLens의 핵심 목표와 기술 스택을 한눈에 정리합니다.

Product Vision
구독 서비스를 수동 등록하고, 갱신 알림 · 지출 분석 · AI 기반 대체재 추천 · 통신사/네이버 번들 최적화를 제공하는 개인 구독 관리 SaaS. 실제 런칭 및 수익화 목표.
Target User
월 3개 이상 구독 서비스를 이용하며 지출을 최적화하고 싶은 한국 사용자. 특히 통신사 요금제와 OTT를 결합해 절약하고 싶은 20~40대.
Architecture
Monorepo · Spring Boot + Next.js (App Router) · PostgreSQL + pgvector · Redis · Docker Compose
Core Differentiator
단순 구독 트래커를 넘어, 통신사 결합/네이버플러스 번들 최적화를 AI가 분석해 실질적인 절약 금액을 제시하는 추천 엔진
Revenue Model
Freemium — Free (5개 제한) / Pro $5~9/월 (무제한 + AI 추천 + 번들 최적화)
⚙️
Backend
Spring Boot · JWT (HttpOnly Cookie) · Redis (Token Cache) · Resend API (Email)
com.sublens.backend
🖥️
Frontend
Next.js 14 (App Router) · TanStack Query · Axios Interceptor · Tailwind CSS
features/ + views/
🗄️
Database
PostgreSQL 16 · pgvector (임베딩 저장) · Redis 7 (세션/캐시) · JSONB (번들 혜택 구조)
postgres:16
🤖
AI / ML
Claude API (추천 리포트 생성) · OpenAI Embeddings (서비스 정보 임베딩) · RAG 파이프라인
pgvector + RAG
🐍
Crawler
Python · Playwright (JS 렌더링 대응) · Spring @Scheduled (배치 트리거) · 매일 새벽 2시 실행
batch / cron
🐳
Infra
Docker Compose · Nginx (Reverse Proxy) · 포트: Backend 8081 · Frontend 3000
ddl-auto: update

핵심 기능

07
🏆 구독 번들 최적화 추천 (AI)
사용자의 등록된 구독 목록을 분석하여, 통신사 결합 상품 또는 네이버플러스 멤버십을 통해 절약 가능한 금액과 최적 구독 조합을 AI가 자동 추천합니다. Claude API가 자연어 형태의 맞춤 리포트를 생성합니다.
Claude API pgvector RAG 번들 매핑 엔진 Pro 전용

"현재 Netflix(월 13,500원), YouTube Premium(월 14,900원), Disney+(월 9,900원)를 개별 구독 중이십니다. → LGU+ 유독Pick 유튜브+디즈니 번들로 전환 시 월 23,800원 → 23,800원 절약 가능 → 네이버플러스 멤버십(4,900원) 가입 시 Netflix 광고형 포함 + 쇼핑 5% 적립"

04 — Roadmap

개발 로드맵

우선순위 기반 7단계 개발 순서. 각 단계는 이전 단계의 데이터를 기반으로 합니다.

Phase 1 · 완료
기반 인프라 & 인증 시스템
Spring Boot + Next.js 모노레포 구성, JWT HttpOnly 쿠키 인증, Redis 토큰 캐시, Docker Compose 환경 세팅 완료.
Spring BootJWT AuthRedisDocker Compose
Phase 2 · 진행중
구독 CRUD & 목록 UI
Subscription 엔티티 ERD 설계, REST API 구현, 프론트엔드 구독 목록 페이지 및 등록/수정 폼 완성.
ERD 설계REST CRUD구독 목록 UI모달 폼
Phase 3
지출 분석 대시보드
월별 / 카테고리별 집계 API, 차트 시각화, 연간 환산 및 낭비 지수 계산 로직 구현.
집계 API차트 컴포넌트낭비 지수
Phase 4
갱신일 알림 시스템
Resend API 기반 갱신 D-3, D-1 자동 이메일. Spring @Scheduled 배치, 알림 이력 관리.
Resend API@Schedulednotification_logs
Phase 5
pgvector 세팅 & 서비스 DB 구축
PostgreSQL pgvector 확장 활성화, 서비스 정보 임베딩 데이터 적재, 유사도 검색 API 개발.
pgvectorOpenAI Embeddings서비스 DB
Phase 6
번들 최적화 추천 엔진
통신사/네이버 번들 크롤러 구축, bundle_products 테이블 적재, 구독 매핑 로직, Claude API 자연어 리포트 생성.
Playwright 크롤러번들 매핑절감액 계산Claude API
Phase 7
AI 챗봇 & 가격 변동 알림
RAG 기반 챗봇 어시스턴트 완성, 구독 서비스 가격 변동 크롤링 및 알림 연동.
RAG 챗봇가격 변동 알림뉴스 크롤링
