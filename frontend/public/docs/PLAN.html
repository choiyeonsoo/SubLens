01 — Overview

프로젝트 개요

SubLens의 핵심 목표와 기술 스택을 한눈에 정리합니다.

Product Vision
구독 서비스를 수동 등록하고, 갱신 알림 · 지출 분석 · AI 기반 대체재 추천 · 통신사/네이버 번들 최적화를 제공하는 개인 구독 관리 SaaS. 실제 런칭 및 수익화 목표.
Target User
월 3개 이상 구독 서비스를 이용하며 지출을 최적화하고 싶은 한국 사용자. 특히 통신사 요금제와 OTT를 결합해 절약하고 싶은 20~40대.
Architecture
Monorepo · Spring Boot + Next.js (App Router) · PostgreSQL + pgvector · Redis · Docker Compose
Core Differentiator
단순 구독 트래커를 넘어, 통신사 결합/네이버플러스 번들 최적화를 AI가 분석해 실질적인 절약 금액을 제시하는 추천 엔진
Revenue Model
Freemium — Free (5개 제한) / Pro $5~9/월 (무제한 + AI 추천 + 번들 최적화)
⚙️
Backend
Spring Boot 3.5 · JWT (HttpOnly Cookie) · Redis (Token + Refresh Rotation) · Resend API
Java 21 · Spring Security 6.5
🖥️
Frontend
Next.js 16 · React 19 · TanStack Query v5 · Axios · Zustand · shadcn/ui · Tailwind CSS
Recharts · Sonner Toast
🗄️
Database
PostgreSQL 16 · pgvector (임베딩) · Redis 7 (세션/캐시) · JSONB (번들 혜택 구조)
postgres:16
🤖
AI / ML
Claude API (추천 리포트 생성) · OpenAI Embeddings (서비스 임베딩) · RAG 파이프라인
pgvector + RAG
🐍
Crawler
