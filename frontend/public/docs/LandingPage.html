구독 관리, 이렇게
달라집니다

복잡하게 흩어진 구독 서비스를 한 곳에서 파악하고,
스마트한 알림으로 불필요한 지출을 방지하세요.

📊

실시간 대시보드

모든 구독의 현황을 직관적인 차트와 카드로 한눈에 확인하세요. 월별 지출 추이와 절감 포인트를 바로 파악할 수 있어요.

🔔

스마트 갱신 알림

청구일 3일, 1일 전 이메일과 푸시 알림을 보내드립니다. 깜빡하고 결제되는 일, 이제 없어요.

📈

지출 패턴 분석

카테고리별 구독 비용을 분석하여 어디에 얼마를 쓰고 있는지 명확하게 보여드립니다.

🔒

보안 인증 시스템

JWT 기반 HttpOnly Cookie 인증과 Refresh Token Rotation으로 안전한 로그인 환경을 제공합니다.

🗂️

카테고리 분류

엔터테인먼트, 업무용 툴, 클라우드 등 카테고리별로 자동 분류되어 관리가 더욱 편리해집니다.

💡

절감 인사이트

사용 패턴을 분석해 비용 절감이 가능한 구독을 알려드립니다. 평균 월 ₩18,000 절약 효과!
