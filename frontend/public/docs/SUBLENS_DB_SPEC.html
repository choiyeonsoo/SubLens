PK 기본키 FK 외래키 UQ 유니크 필수 관계 (NOT NULL FK) 선택 관계 (nullable FK)
  • subscriptions.user_id → users.id  (1:N, 논리적 FK)
  • subscriptions.category_id → subscription_categories.id  (N:1, nullable — 점선)
  • subscription_services.category_id → service_categories.id  (N:1, NOT NULL)
  • subscription_categories ≠ service_categories: 전자는 사용자 구독 분류, 후자는 서비스 카탈로그 분류

5. DDL 스크립트

Hibernate ddl-auto: update가 자동 관리하지만, 초기 구성 참조용입니다.

-- users
CREATE TABLE users (
    id           UUID PRIMARY KEY,
    email        VARCHAR NOT NULL UNIQUE,
    password_hash VARCHAR NOT NULL,
    name         VARCHAR NOT NULL,
    phone_number VARCHAR,
    role         VARCHAR NOT NULL DEFAULT 'USER',
    created_at   TIMESTAMP NOT NULL,
    updated_at   TIMESTAMP
);

-- subscription_categories
CREATE TABLE subscription_categories (
    id         UUID PRIMARY KEY,
    name       VARCHAR(50) NOT NULL UNIQUE,
    icon       VARCHAR(100),
    color      VARCHAR(7),
    is_default BOOLEAN NOT NULL DEFAULT FALSE
);

-- subscriptions
CREATE TABLE subscriptions (
    id                  UUID PRIMARY KEY,
    user_id             UUID NOT NULL,
    category_id         UUID REFERENCES subscription_categories(id),
    service_name        VARCHAR(100) NOT NULL,
    description         TEXT,
    amount              NUMERIC(12,2) NOT NULL,
    currency            VARCHAR(10) NOT NULL DEFAULT 'KRW',
    billing_cycle       VARCHAR(10) NOT NULL DEFAULT 'MONTHLY',
    start_date          DATE,
    next_billing_date   DATE NOT NULL,
    status              VARCHAR(10) NOT NULL DEFAULT 'ACTIVE',
    notify_before       BOOLEAN NOT NULL DEFAULT TRUE,
    notify_days_before  SMALLINT NOT NULL DEFAULT 3,
    created_at          TIMESTAMP NOT NULL,
    updated_at          TIMESTAMP
);

-- service_categories
CREATE TABLE service_categories (
    id            UUID PRIMARY KEY,
    name          VARCHAR(50) NOT NULL UNIQUE,
    display_order INTEGER NOT NULL
);

-- subscription_services
CREATE TABLE subscription_services (
    id            UUID PRIMARY KEY,
    name          VARCHAR(100) NOT NULL,
    category_id   UUID NOT NULL REFERENCES service_categories(id),
    logo_domain   VARCHAR(100) NOT NULL,
    display_order INTEGER NOT NULL,
    is_active     BOOLEAN NOT NULL DEFAULT TRUE,
    created_at    TIMESTAMP NOT NULL
);

6. 운영 참고사항

6.1 ddl-auto: update 주의

  • 컬럼 추가는 자동으로 적용됩니다.
  • 컬럼 삭제/타입 변경은 자동으로 반영되지 않으며, 수동 ALTER TABLE이 필요합니다.
  • 예: currency 컬럼의 커스텀 타입 currency_code를 VARCHAR로 변경 시 수동 마이그레이션 필요했음.
