# SUBLENS 사용 가이드

> 마지막 업데이트: 2026-04-03

---

## 1. SUBLENS란?

SUBLENS는 내가 구독 중인 모든 서비스를 한 곳에서 관리할 수 있는 구독 관리 플랫폼입니다.

**주요 기능**

| 기능      | 설명                                                   |
| --------- | ------------------------------------------------------ |
| 구독 관리 | 넷플릭스·유튜브·스포티파이 등 개별 구독 등록·수정·삭제 |
| 번들 구독 | LGU+ 유독, SKT T우주 등 통신사 번들 상품 통합 관리     |
| 지출 분석 | 월별·카테고리별 구독료 시각화 및 연간 예측             |
| AI 추천   | 구독 최적화·비용 절감 방법을 AI가 대화형으로 안내      |
| 갱신 알림 | 결제일 N일 전 미리 알림 설정                           |

## 2. 회원가입 / 로그인

### 회원가입

1. 우상단 **회원가입** 버튼 클릭
2. 아래 정보를 입력합니다.

   | 항목          | 설명                                                |
   | ------------- | --------------------------------------------------- |
   | 이메일        | 로그인에 사용할 이메일 주소                         |
   | 비밀번호      | 영문·숫자 조합 권장                                 |
   | 비밀번호 확인 | 위와 동일하게 입력                                  |
   | 이름          | 실명                                                |
   | 휴대폰 번호   | 연락처                                              |
   | 통신사        | SKT · KT · LGU+ 중 선택 (AI 번들 추천에 활용됩니다) |

3. **가입하기** 클릭 → 가입 완료

> **통신사를 정확히 선택**하면 내 통신사에서 가입 가능한 번들 상품만 추천받을 수 있습니다.

### 로그인

1. 이메일과 비밀번호 입력 후 **로그인** 클릭
2. 로그인 상태는 자동으로 유지됩니다.

### 비밀번호 재설정

1. 로그인 화면에서 **비밀번호를 잊으셨나요?** 클릭
2. 가입한 이메일 주소 입력 → **재설정 링크 보내기**
3. 이메일로 받은 링크 클릭
4. 새 비밀번호 입력 후 완료

> 재설정 링크는 **10분간** 유효합니다.

## 3. 구독 추가하는 방법

구독 목록 페이지(`/subscriptions`) 우상단의 **+ 구독 추가** 버튼을 클릭합니다.

### 개별 구독 추가

가장 일반적인 방법입니다. 넷플릭스, 유튜브 프리미엄처럼 단독으로 결제하는 구독에 사용합니다.

1. 모달 상단의 탭에서 **개별 구독** 선택 (기본값)
2. 아래 항목을 입력합니다.

   | 항목        | 필수 | 설명                                                |
   | ----------- | ---- | --------------------------------------------------- |
   | 서비스 이름 | ✅   | 직접 입력하거나 자동완성 목록에서 선택 (최대 100자) |
   | 카테고리    | —    | 엔터테인먼트·음식배달 등 분류 선택                  |
   | 금액        | ✅   | 결제 금액 (0 이상)                                  |
   | 통화        | ✅   | KRW · USD · EUR · JPY · GBP                         |
   | 결제 주기   | ✅   | 월간 · 주간 · 연간                                  |
   | 시작일      | —    | 기본값: 오늘                                        |

3. **결제 주기별 추가 항목**

   | 주기     | 추가 입력            |
   | -------- | -------------------- |
   | **월간** | 매월 결제일 (1~31일) |
   | **주간** | 결제 요일 (월~일)    |
   | **연간** | 결제 월 + 일         |

4. **갱신 알림 설정**
   - 알림 없음 / 1일 전 / 3일 전 / 7일 전 / 직접 입력(1~30일) 중 선택

5. **저장** 클릭 → 구독 목록에 추가됩니다.

### 번들 구독 추가

LGU+ 유독, SKT T우주 등 통신사 번들에 포함된 서비스를 추가할 때 사용합니다.

1. 모달 상단 탭에서 **번들 구독** 선택
2. **번들 제공사** 선택

   | 제공사        | 설명                                 |
   | ------------- | ------------------------------------ |
   | LGU+ 유독     | LGU+ 고객 전용 OTT·음악 등 구독 번들 |
   | SKT T우주     | SKT 고객 전용 구독 패키지            |
   | KT 구독       | KT 고객 전용 구독 서비스             |
   | 네이버 플러스 | 통신사 무관 네이버 멤버십 기반 번들  |

3. **번들 상품** 드롭다운에서 가입 중인 플랜 선택
   - 선택하면 포함된 서비스 목록이 태그로 표시됩니다.
4. 금액, 결제 주기 등 나머지 항목 입력
5. **저장** 클릭

> 번들로 등록된 구독은 AI 추천 시 중복 추천 대상에서 제외되어 정확한 최적화 결과를 받을 수 있습니다.

## 4. 구독 목록 관리

### 목록 보기 (`/subscriptions`)

- **검색**: 서비스 이름으로 필터링
- **상태 필터**: 전체 · 활성 · 일시정지 · 취소됨
- **정렬**: 최신순 · 다음 결제일 순 · 금액 높은순

### 구독 수정

카드 우측의 **편집** 아이콘 클릭 → 등록 시와 동일한 폼에서 수정 후 저장

### 구독 상태 변경

카드에서 상태 버튼 클릭으로 전환 가능합니다.

| 상태         | 설명                                    |
| ------------ | --------------------------------------- |
| **활성**     | 현재 결제 중인 구독                     |
| **일시정지** | 일시적으로 사용 중단 (비용 추적은 유지) |
| **취소됨**   | 해지한 구독 (기록 보존)                 |

### 구독 삭제

카드의 **삭제** 아이콘 클릭 → 확인 → 완전히 삭제됩니다.

> 삭제된 데이터는 복구할 수 없습니다. 해지한 경우라도 기록 보존을 위해 **취소됨** 상태로 두는 것을 권장합니다.

## 5. 지출 분석 보기

분석 페이지(`/analytics`)에서 구독 지출 현황을 확인합니다.

### 상단 요약 카드 (4개)

| 카드         | 내용                       |
| ------------ | -------------------------- |
| 총 누적 지출 | 전체 기간 동안 지출한 총합 |
| 월 평균      | 월별 평균 구독료           |
| 전월 대비    | 지난달과 이번달 증감 비율  |
| 연간 예상    | 현재 구독 기준 1년 예측액  |

### 기간 필터

상단 탭에서 조회 기간을 선택합니다: **3개월 · 6개월 · 1년 · 전체**

### 차트 및 분석

| 섹션               | 내용                                             |
| ------------------ | ------------------------------------------------ |
| **월별 지출 추이** | 막대 그래프로 월별 지출 변화                     |
| **카테고리 분포**  | 도넛 차트로 카테고리별 비중                      |
| **서비스 순위**    | 지출 금액 상위 서비스 목록 및 비율               |
| **기간 비교**      | 이번달 · 지난달 · 3개월 평균 · 최고/최저 월 비교 |

### 대시보드 (`/dashboard`)

메인 화면에서는 간략한 현황을 확인할 수 있습니다.

- 이번 달 지출 · 이번 주 지출 · 활성 구독 수 · 연간 예상
- 이번 달 및 다음 달 갱신 예정 구독 달력
- 카테고리별 지출 분포
- D-day 배지: 결제일까지 남은 일수 (🔴 3일 이내 · 🟡 7일 이내 · 🟢 그 외)

## 6. AI 추천 기능 사용법

AI 추천 페이지(`/recommend`)에서 구독 관련 무엇이든 자연어로 질문할 수 있습니다.

### 화면 구성

- **왼쪽 패널**: 과거 대화 목록 (최대 20개 세션 저장)
- **오른쪽 채팅 영역**: AI와 대화

### 질문 예시

화면 중앙의 빠른 질문 버튼을 눌러 바로 시작하거나, 직접 입력할 수 있습니다.

```
이번 달 구독료 총액이 얼마야?
내 구독 최적화해줘
유독 해지하면 위약금 있어?
구독 어떻게 추가해?
넷플릭스랑 유튜브 프리미엄 번들이 나한테 이득이야?
```

### 답변 유형 설명

AI 답변에는 아래와 같은 유형 배지가 표시됩니다.

| 배지            | 색상 | 설명                                                            |
| --------------- | ---- | --------------------------------------------------------------- |
| **내 데이터**   | 파랑 | 내 구독 DB를 조회해 수치 기반으로 답변 (예: 월 총액, 연간 지출) |
| **서비스 안내** | 회색 | SUBLENS 사용 방법 안내 (예: 구독 추가하는 법)                   |
| **번들 문의**   | 주황 | 번들 상품 약관·조건 안내 (예: 위약금, 가족 공유 한도)           |
| **구독 최적화** | 보라 | 내 구독과 번들 상품을 비교해 최적 조합 추천                     |

### 최적화 추천 결과 보기

**구독 최적화** 답변은 구조화된 카드로 표시됩니다.

- **절약 배너**: 현재 월 지출 → 최적화 후 예상 지출 및 절감액
- **추천 번들**: 내 구독과 겹치는 번들 상품 (추천 순위, 포함 서비스, 통신사 전용 여부)
- **유지 구독**: 번들로 대체되지 않는 구독 목록
- **주의사항**: 약정 조건, 통신사 전용 안내 등

> AI가 계산한 절감액은 Python 계산 엔진의 결과값이며 Claude AI는 주의사항 생성에만 관여합니다. 따라서 동일 질문에 항상 같은 수치를 제공합니다.

### 대화 세션 관리

- **새 대화**: 왼쪽 패널 상단 **새 대화** 버튼
- **이전 대화 불러오기**: 왼쪽 패널에서 세션 클릭
- **대화 삭제**: 세션에 마우스 오버 → 휴지통 아이콘 클릭
- 대화 내용은 브라우저 로컬스토리지에 저장되며 최대 20개까지 유지됩니다.

## 7. 알림 설정

### 구독별 갱신 알림

각 구독을 추가하거나 수정할 때 **갱신 알림** 항목에서 설정합니다.

| 옵션      | 설명                         |
| --------- | ---------------------------- |
| 알림 없음 | 알림 비활성화                |
| 1일 전    | 결제 전날 알림               |
| 3일 전    | 결제 3일 전 알림 (기본값)    |
| 7일 전    | 결제 1주일 전 알림           |
| 직접 입력 | 1~30일 사이 원하는 날짜 지정 |

> 알림 채널(이메일, 앱 푸시 등) 설정 페이지는 현재 준비 중입니다.

## 8. 자주 묻는 질문 (FAQ)

### 무료 플랜 구독 한도는?

현재 SUBLENS는 등록 가능한 구독 수에 별도 제한을 두고 있지 않습니다. AI 추천 대화는 브라우저당 최대 20개의 세션이 저장됩니다.

---

### 결제 수단 변경은?

SUBLENS는 구독 **정보를 기록·분석**하는 서비스입니다. 실제 구독 결제는 각 서비스(넷플릭스, 유튜브 등)에서 직접 관리합니다. SUBLENS에서는 금액 정보만 업데이트하면 됩니다.

---

### 구독 삭제는?

1. **구독 목록** 페이지 이동
2. 삭제할 구독 카드에서 **삭제** 아이콘 클릭
3. 확인 메시지에서 **삭제** 선택

> 삭제 대신 **취소됨** 상태로 변경하면 지출 이력을 보존하면서 활성 목록에서 제외할 수 있습니다.

---

### 번들 구독 등록 방법은?

1. **+ 구독 추가** 버튼 클릭
2. 모달 상단 탭에서 **번들 구독** 선택
3. 내 통신사에 해당하는 **번들 제공사** 선택 (LGU+ 유독 / SKT T우주 / KT 구독 / 네이버 플러스)
4. **번들 상품** 드롭다운에서 가입 중인 플랜 선택
5. 월 결제 금액과 결제일 입력 후 **저장**

번들로 등록된 구독은 AI 최적화 분석에서 이미 가입된 번들로 인식되어 중복 추천을 방지합니다.

---

### AI 추천이 느리거나 오류가 날 때는?

- AI 서비스는 최대 **30초** 응답 대기 시간이 있습니다.
- 30초를 초과하면 "응답 시간이 초과됐습니다." 메시지가 표시됩니다.
- AI 서버에 연결할 수 없을 경우 "AI 서버에 연결할 수 없습니다." 메시지가 표시됩니다.
- 이 경우 잠시 후 다시 시도해 주세요.

---

### 통화(Currency)를 USD나 JPY로 설정한 경우 분석에 반영되나요?

네. 분석 페이지에서는 KRW가 아닌 통화도 원래 금액 그대로 집계에 포함됩니다. JPY와 KRW는 금액이 클 경우 `k` 단위로 표시됩니다(예: 1,200,000원 → 1,200k).

> 현재 환율 변환 기능은 지원하지 않습니다. 금액은 입력한 값 그대로 사용됩니다.
